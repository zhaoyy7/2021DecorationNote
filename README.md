## 一、写在前面
自2021年2月25日拿到产证，在经历了将近5个月的漫长放款流程后，终于看到了交房的希望，赶紧装修搞起来~~

家装第一步，当然是选择一个靠谱的装修公司啦。
  首先，装修类型可以分为以下三类，考虑到我们的实际情况，选择了半包，既不会花费太多精力，又可以自己掌控花销。
  1. 清包: 自己找工程队、自己采购所有材料。
  2. 半包: 自己包主材（瓷砖，橱柜），装修队包人工、辅材。
  3. 全包: 装修公司全包所有内容。

找装修公司就像找工作，前期可以在市场上随便看看，了解下行情。然而最心仪的总是要放在最后考察。前期考察了2家公司学习了一些装修知识后，我们便直奔同济装潢签单，原因有二：
  1. 口碑好，同事之前也用的同济，后期反馈很好。
  2. 设计师靠谱，接待我们的设计师是哈工大土木系毕业生，沟通顺畅。

选好装修公司后，接下来按照流程一步步来就好啦。这件事情就像写代码，架构师已经搭好了框架，接下来就开始精雕细琢各个功能模块。

## 四、经验教训
1. 业主方要多在装修微信大群里问下进度，以保证工期。有需要多方保持透明的问题也在微信群里抛出来，避免日后责任界定不清。
2. 轻易不要得罪设计师和工长，这两个人至关重要。还有物业，也是得罪不起的角色。保证活干好的前提下，尽量让大家都开心吧。
3. 经常去工地看看，戴好口罩，穿长裤。去的时候带包烟发一发（不要太贵），夏天的话带点冰饮料给工人，都不容易。
4. 装修寄家里的材料，收件人一律写工长。
5. 水电图一定要有，方案改动也要新出水电图，不要不好意思，否则水电定位的时候你会一脸懵逼。
6. 一定要瓷砖供应商亲自上门测量下到底需要多少砖，下订单的时候就谈好，不要不好意思。
7. 有些东西线下比线上便宜，比如施耐德的开关插座，卡萨帝的壁挂洗衣机。
8. 线下买东西一定要去多家供应商集中的地方，比如漕溪路好饰家，下午他们快下班的时候比较容易砍价。
9. 窗户一定要供应商量好尺寸再去谈价交钱。可以拿着别人家测量图纸去要报价，但不要根据这个尺寸付钱，不要问我怎么知道的。
